    What is this?

    You are looking at a code browser for test.

    By using this web interface, you can navigate the source code of test by following simple links, search it by using the box in the navigation bar, or use vi inspired key bindings to move within files.

    It should all be pretty intuitive, but to get started, here are a few things you may want to try:

    • Search for files or symbols by using 'f', 's' or the pull down menu to the left of the search box.
    • Press '?' to have a help screen with key bindings, and 'a' or ESC to hide this screen.
    • Try to drag variables or functions to get more details about them.

    This site was generated via sbexr, which uses LLVM and clang to parse and index the code.

    sbexr is free software (as in "free speech"), under heavy development. sbexr.rabexc.org and the Linux kernel source code are used as a playground to test new features, observe bugs, and gather feedback. Check there often if you want to see new features in action.

    As of today, the best way to know more about the project or participate in the development is to join the mailing list.
